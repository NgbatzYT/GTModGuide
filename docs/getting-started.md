# Getting Started

### Requirements

To get started with Gorilla Tag modding, you only need:

- **PCVR Gorilla Tag**  
  You can purchase it from:  
  - [Steam (recommended)](https://steampowered.com)  
  - [Meta PC app](https://www.meta.com/help/quest/1517439565442928/)  

  The game costs $20.
